# PLTE for MEMS Acceleration Data

## Overview
This repository provides the implementation of the **Piecewise-Linear Trend Extraction (PLTE)** method used in this study.

The code demonstrates how PLTE can be applied to MEMS acceleration time series under **irregular sampling conditions** and in the presence of **local missing segments**, without requiring interpolation or resampling.

---

## Key Features
- Piecewise-linear trend representation based on sparse changepoints  
- Direct operation on original timestamps (no resampling)  
- Robust behavior across local missing-data intervals  
- Comparison with a representative recursive smoothing approach (Kalman filter)  

---

## Minimal Example
The provided example allows reproduction of representative behaviors discussed in the paper:
